# uniapp-starter-template

uniapp 小程序启动模板

- vue3
- eslint + prettier
- vant
- pinia
- vite
- unocss
- husky + lint-staged
- pnpm
- echarts

### 安装依赖

```
pnpm i
```

### 启动

具体需要打开哪一个，可看 package.json 中的 scripts

```
pnpm dev:mp-weixin
```

### 微信开发者工具

1. 在 src/manifest.json 中的 appid 字段填写自己的 appid
2. 将 dist/mp-weixin 目录添加进微信开发者工具

然后就可以开发了！！！
